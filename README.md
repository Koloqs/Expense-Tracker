# Expense-Tracker
A simple expense tracker to manage your finances.

CLI followed by restrictions from https://roadmap.sh/projects/expense-tracker :

## Usage

```bash
python expense_tracker.py add --description "Lunch" --amount 20
python expense_tracker.py list
python expense_tracker.py summary
python expense_tracker.py delete --id 1
python expense_tracker.py delete --all
```
